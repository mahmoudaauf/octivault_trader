## How Trade Outcomes Drive Learning

```
Trade #1: BUY BTCUSDT
Entry: $92,435
Exit: $92,780
Result: +0.37% ✅ WIN

→ Model learns: "This signal worked!"
→ Confidence increased in similar patterns
→ Win rate: 1/1 (100%)

Trade #2: BUY ETHUSDT
Entry: $3,450
Exit: $3,425
Result: -0.72% ❌ LOSS

→ Model learns: "This signal was wrong"
→ Confidence decreased in similar patterns
→ Win rate: 1/2 (50%)

Trade #3+: Models know which signals work best
→ System gets smarter with each trade
→ Eventually reaches optimal confidence threshold
```
