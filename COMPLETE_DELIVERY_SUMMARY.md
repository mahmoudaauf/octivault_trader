## 🎓 Key Concepts Explained

### 1. NAV-Based Strategy Selection
```python
if nav < $500:
    use_maker_orders = True  # Small account: fees matter
else:
    use_maker_orders = False # Large account: speed matters
```

Your $100 account qualifies for maker orders! ✓

### 2. Spread-Based Pricing
```
Bid:  100.00
Ask:  100.05
Spread: 0.05

Maker limit price = bid + (ask - bid) × 0.2
                  = 100.00 + 0.05 × 0.2
                  = 100.01  ← Inside spread, better than market!
```

### 3. 5-Second Timeout Logic
```
Place limit order → Wait 5 seconds
├─ If filled:      Success! Cost = 0.03%
└─ If timeout:     Cancel + place market at current price
                   Graceful fallback, execution guaranteed
```

### 4. Universe Optimization
```
Current:  $100 ÷ 53 symbols = $1.89/symbol (too small!)
Optimized: $100 ÷ 5 symbols = $20/symbol (proper sizing!)

Benefits:
├─ 10x larger positions
├─ Better fill probability

```
